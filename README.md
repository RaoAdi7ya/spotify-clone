🎧 Spotify Clone UI (HTML + CSS)

A clean, responsive Spotify-inspired web player interface built using HTML5 and CSS3.

⚠️ This project is a front-end UI clone created for learning and portfolio purposes. It does not include real audio playback, authentication, or Spotify API integration.

🚀 Preview

This project recreates the familiar Spotify layout:

🎵 Left sidebar navigation + library section
📀 Main content area with playlist/album cards
🔝 Sticky top navigation bar
▶️ Fixed bottom music player with controls & sliders
🛠️ Tech Stack
HTML5
CSS3
Font Awesome (for icons)
Google Fonts (Montserrat)

✨ Features
🎨 Spotify-like dark theme UI
📱 Responsive design for smaller screens
🧩 Reusable card components
🎚️ Custom styled sliders (progress & volume)
